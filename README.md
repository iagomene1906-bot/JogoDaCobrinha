## 📖 Sobre o Projeto

Este é um clone do clássico "Jogo da Cobrinha" (Snake), desenvolvido inteiramente com a linguagem **C#** e a interface gráfica do **Windows Forms (WinForms)**. 

O objetivo principal deste projeto foi aplicar e consolidar conceitos fundamentais de Programação Orientada a Objetos (POO), manipulação de eventos de teclado, renderização gráfica básica e lógica de jogos em ambientes desktop.

## ✨ Funcionalidades

- Movimentação em 4 direções (Cima, Baixo, Esquerda, Direita).
- Sistema de pontuação: a cobra cresce e a pontuação aumenta ao capturar a "comida".
- Detecção de colisões (Game Over ao bater nas bordas da tela ou no próprio corpo).
- Interface simples e responsiva aos comandos do jogador.

## 🚀 Tecnologias Utilizadas

- **C#**
- **.NET (Windows Forms)**
- **Visual Studio** (Ambiente de Desenvolvimento)

## 🛠️ Como Executar o Projeto

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina a seguinte ferramenta:
- [Visual Studio](https://visualstudio.microsoft.com/pt-br/) (com a carga de trabalho de desenvolvimento para Desktop com .NET instalada).

### Passos para rodar
1. Clone este repositório:
```bash
   git clone https://github.com/iagomene1906-bot/JogoDaCobrinha.git
```
2. Abra o arquivo da solução (`JogoDaCobrinha.sln`) no seu Visual Studio.
3. No painel superior, certifique-se de que o projeto de inicialização está correto e clique em **Iniciar** (ou pressione `F5`).
4. O jogo será aberto em uma nova janela. Divirta-se!
